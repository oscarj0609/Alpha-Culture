Alpha Culture-Proyecto web desarrollado como parte del proyecto #1 de la asignatura Proyecto Integrador Web.
Alpha Culture es una webzine temática enfocada en cultura fitness, mentalidad enfocada y entrenamientos personalizados. 
Incluye una sección hero, artículos informativos, galería de imágenes, un carrusel interactivo desarrollado únicamente con HTML y CSS grid, una columna lateral con biografía/autor/links, y un pie de página con formulario de suscripción.
